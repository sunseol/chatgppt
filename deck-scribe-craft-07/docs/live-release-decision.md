## Decision basis

The current worktree has stronger Live-readiness contracts than the previous mock-provider build: production mock selection is blocked by policy, provider provenance and structured Codex output gates exist, research fetches preserve capture metadata, image provider responses preserve request metadata, live generation report lineage is checked per slide, Golden Path E2E and manual QA evidence bundles have local verifiers, DF-247 final release gate criteria are encoded, and the final export gate rejects contaminated production lineage.
